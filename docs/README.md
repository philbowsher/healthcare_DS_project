# Documentation

This directory contains documentation for the European Healthcare Data Analysis project.

## Available Documentation

### User Guides
- **Getting Started**: Installation and basic usage
- **Data Sources**: Information about data sources and how to access them
- **Analysis Guide**: How to perform different types of analysis

### Technical Documentation
- **API Reference**: Documentation for all modules and functions
- **Data Schema**: Structure and format of datasets

### Examples
- **Example Analyses**: Sample analysis workflows
- **Visualization Gallery**: Examples of available visualizations

## Contributing to Documentation

When adding new features or making changes:
1. Update relevant documentation files
2. Add examples if introducing new functionality
3. Keep documentation in sync with code changes
